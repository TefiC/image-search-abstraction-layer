My API Project #4: Image Search Abstraction Layer

How it works:
    1) You can pass a string as a parameter, and it will find images that match that query, and links you can click on to access the image and it's context (example: dog)
    2) You must define the number of results you wish to find setting ?offset='integer'
    2) You can get the most recent queries by using https://back-end-projects-freecodecamp.herokuapp.com/image-search-abstraction-layer/recent-searches/

Example usage:

https://back-end-projects-freecodecamp.herokuapp.com/image-search-abstraction-layer/new-search/dog?offset=2
